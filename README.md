# Tic Tac Toe JS

A simple Tic Tac Toe game built using **HTML, CSS, and JavaScript** for frontend practice.

## Overview
This project is a basic Tic Tac Toe game that allows two players to play alternately on the same device. The game highlights winning combinations and allows players to restart the game after completion.

## Features
- 2-player Tic Tac Toe game
- Highlights winning combination
- Reset button to start a new game
- Responsive design for desktop and tablet screens

## Tech Stack
- HTML
- CSS
- JavaScript

## Notes
- This project is **for learning and practice purposes only**.
- No backend or online multiplayer functionality included.
